### Cześć! 👋
Zbiór projektów opartych na mikrokontrolerze ESP8266, realizujących zadania z zakresu inteligentnego domu, komunikacji bezprzewodowej oraz integracji urządzeń z bazami danych.

Moje projekty ewoluowały przez trzy kluczowe etapy:
- Client-to-Web: Odczyt danych z globalnych serwisów pogodowych.
- Local Control: Tworzenie własnych serwerów i punktów dostępowych (AP) do sterowania domem.
- Data Persistence: Pełna integracja z backendem (PHP) i relacyjną bazą danych (SQL) w celu archiwizacji pomiarów.
